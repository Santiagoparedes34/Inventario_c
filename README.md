 Sistema de Inventario en C

Descripción

Este proyecto es un sistema de inventario desarrollado en lenguaje C que funciona mediante un menú en consola.
El programa permite ingresar productos con sus precios y realizar diferentes operaciones para analizar la información del inventario.

 Funcionalidades del sistema

El programa muestra un menú con las siguientes opciones:

1. Ingresar productos y precios
2. Calcular el precio total del inventario
3. Mostrar el producto de mayor y menor precio
4. Calcular el promedio de los precios
5. Buscar un producto
6. Salir del programa

## Estructura del proyecto

* **main.c** → Archivo principal donde se ejecuta el programa
* **funciones.c** → Implementación de las funciones del sistema
* **funciones.h** → Declaración de las funciones utilizadas

Cómo compilar el programa


gcc main.c funciones.c -o inventario


Cómo ejecutar el programa

./inventario


Tecnologías utilizadas

* Lenguaje C
* Programación estructurada
* Uso de funciones y archivos `.h`
